# AI Product + Data Portfolio

This portfolio demonstrates how I use AI, Python, and Git to analyze business problems, create repeatable workflows, and communicate insights through structured technical documentation.

## Portfolio Goals

- Use Git like a real developer and product analyst.
- Apply AI tools strategically, with clear prompts, validation, and reflection.
- Analyze data and explain the business value behind technical work.
- Document process, decisions, outputs, and lessons learned.
- Create reusable artifacts: code, notebooks, prompts, recommendations, workflows, and reflections.

## How To Review This Portfolio

1. Start with `portfolio-overview.md` for the overall story.
2. Review `projects/01-foodhub-eda/` for the strongest technical project.
3. Review `projects/02-ai-prompt-engineering/` to see how AI was used intentionally.
4. Review `projects/03-ai-business-use-case/` for product strategy and workflow design.
5. Review `projects/05-personal-hobby-widgets/` for the creative personal section.
