# AI Prompt Engineering Portfolio

## Objective

This project shows how AI can be used strategically to support data analysis, business thinking, documentation, and presentation work.

## Goal

The goal is to demonstrate prompting, iteration, evaluation, and validation. Strong AI use is not just asking a broad question. It means defining the role, task, context, constraints, and output format.

## Artifacts

- `prompt-library.md`: reusable prompts organized by analysis task
- `before-after-examples.md`: examples of weak prompts improved into stronger prompts
- `reflection.md`: explanation of how AI was used responsibly

## Skills Demonstrated

- Prompt design
- AI-assisted data analysis
- Critical thinking
- Output evaluation
- Documentation
- Business communication

## Evaluation Approach

AI outputs were evaluated by checking:

- whether the response answered the specific question
- whether assumptions were stated clearly
- whether the output could be verified with data
- whether the recommendation made business sense
- whether the final wording was clear for the intended audience
