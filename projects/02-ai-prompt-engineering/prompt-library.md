# AI Prompt Library

## Prompt 1: Data Cleaning Support

### Original Prompt

"Help me clean this data."

### Improved Prompt

"Act as a data analyst. Review this FoodHub dataset and identify potential data quality issues including missing values, duplicate order IDs, incorrect data types, inconsistent categorical values, and outliers. Explain why each issue matters before suggesting Python code."

### Why This Is Better

The improved prompt gives the AI a role, a clear task, specific checks, and an expected explanation style.

## Prompt 2: Exploratory Data Analysis Planning

### Original Prompt

"What charts should I make?"

### Improved Prompt

"Act as a product analyst working on a food delivery dataset. Recommend charts for analyzing restaurant demand, cuisine popularity, delivery time, preparation time, order cost, and customer ratings. For each chart, explain the business question it answers and what decision it could support."

### Why This Is Better

The improved prompt connects chart choices to business questions instead of creating visuals without a purpose.

## Prompt 3: Business Recommendation Support

### Original Prompt

"Write recommendations from my analysis."

### Improved Prompt

"Use the following FoodHub findings to write 3-5 business recommendations for a food delivery operations team. Separate recommendations from observations, avoid overstating causation, and explain the expected business value of each recommendation."

### Why This Is Better

The improved prompt asks the AI to distinguish evidence from action and to avoid unsupported conclusions.

## Prompt 4: Code Debugging

### Original Prompt

"Why is my code broken?"

### Improved Prompt

"Act as a Python tutor. I am getting this error in a Pandas notebook. Explain the likely cause in plain language, identify the exact line that needs attention, and suggest the smallest code change that would fix it. Do not rewrite the whole notebook unless necessary."

### Why This Is Better

The improved prompt limits the scope and asks for an explanation that helps the user learn.

## Prompt 5: Presentation Support

### Original Prompt

"Make slides for this."

### Improved Prompt

"Turn these FoodHub EDA findings into a 5-slide presentation for a non-technical audience. Use one key message per slide, include speaker notes, and focus on business impact rather than technical details."

### Why This Is Better

The improved prompt defines the audience, format, length, and communication goal.

## Prompt 6: Critical Thinking And Assumption Checking

### Original Prompt

"Is this analysis good?"

### Improved Prompt

"Review this FoodHub analysis as a skeptical product analytics manager. Identify unsupported assumptions, missing data quality checks, possible alternative explanations, and recommendations that need stronger evidence."

### Why This Is Better

The improved prompt asks AI to challenge the work rather than only polish it.
