# Prompt Engineering Reflection

## What I Learned

Prompt engineering is less about finding a magic phrase and more about giving the AI enough context to produce useful work. The best prompts define the role, task, data context, constraints, and expected output.

## How I Used AI Strategically

I used AI to support the work in targeted ways:

- generate analysis questions
- identify data quality checks
- explain Python concepts
- improve documentation
- draft business recommendations
- challenge assumptions and surface limitations

## How I Validated AI Output

I checked AI-generated output against the actual dataset, reviewed whether recommendations were supported by evidence, and revised wording when conclusions sounded stronger than the data allowed.

## Key Takeaway

AI is most useful when it is treated like an assistant that can accelerate thinking, not as an authority that automatically provides the final answer.
