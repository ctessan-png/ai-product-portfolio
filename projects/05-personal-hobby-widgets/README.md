# Personal Hobby Widgets

## Objective

This section adds a fun, personal layer to the portfolio by turning hobbies and interests into small interactive code artifacts.

The goal is to show that technical projects can be both useful and personal. Each widget is designed to connect a hobby to a product or data skill such as recommendation logic, simple state management, user experience design, or AI-assisted ideation.

## Current Widgets

| Widget | Purpose | Skill Shown |
| --- | --- | --- |
| Hobby Idea Generator | Generates small project ideas based on selected interests | Product ideation, prompt framing, UI logic |
| Mood Board Planner | Turns a theme into content, data, and AI project ideas | Creative strategy, structured thinking |
| Personal Portfolio Cards | Presents hobbies as portfolio-ready project concepts | Communication and positioning |

## How To Use

Open `widgets/index.html` in a browser.

## How This Supports The Portfolio

This project shows that I can:

- translate personal interests into project ideas
- build lightweight interactive artifacts
- connect creativity with practical product thinking
- use AI as a brainstorming partner while still structuring the final output
- communicate technical work in an approachable way

## Personalization Notes

The current widget uses sample hobby categories. Replace them with personal hobbies, such as:

- cooking or restaurants
- music
- travel
- fitness
- fashion
- reading
- real estate
- finance
- community work
- sports

Each hobby can become a small project that demonstrates data, AI, product, or storytelling skills.
