# Widget Ideas

## 1. Restaurant Explorer

Use a favorite food or restaurant interest to create a mini recommendation widget.

### Possible Features

- Filter by cuisine
- Rank by rating, cost, or delivery time
- Generate a short recommendation summary
- Explain why a restaurant is a good fit

### Skills Demonstrated

- Data filtering
- Recommendation logic
- Business explanation
- User-centered design

## 2. Travel Decision Helper

Create a widget that compares travel destinations based on budget, weather, activities, and trip style.

### Possible Features

- Destination comparison table
- Trip mood selector
- AI-generated itinerary prompt
- Budget category tags

### Skills Demonstrated

- Product requirements
- Decision support
- Structured prompt design
- Visual communication

## 3. Music Mood Dashboard

Create a dashboard that maps favorite songs or playlists to moods, activities, or productivity goals.

### Possible Features

- Mood filters
- Playlist categories
- Listening pattern notes
- AI prompt for playlist summaries

### Skills Demonstrated

- Categorization
- Data storytelling
- Personalization logic
- Human-centered AI

## 4. Fitness Or Wellness Tracker

Create a lightweight tracker that connects habits to goals.

### Possible Features

- Weekly habit checklist
- Goal progress indicators
- Reflection prompts
- Simple trend summaries

### Skills Demonstrated

- Metrics design
- Habit tracking
- Interface design
- Reflection and iteration

## 5. Reading Or Learning Tracker

Create a widget that tracks books, articles, courses, or concepts learned.

### Possible Features

- Topic tags
- Notes and takeaways
- AI-generated discussion questions
- Progress by category

### Skills Demonstrated

- Knowledge organization
- Summarization
- Learning analytics
- Documentation
