# Research Notes

## Research Focus

The research phase is focused on understanding what kinds of iPhone-style widgets could become portfolio artifacts.

This includes:

- visual inspiration
- personal hobbies
- widget functionality
- AI support
- technical feasibility
- portfolio value

## Early Observations

Pinterest-style widgets often share a few qualities:

- soft colors
- clean typography
- simple layouts
- mood-based visuals
- personal quotes or reminders
- calendar, habit, or lifestyle themes
- aesthetic consistency across multiple widgets

The strongest widget concepts should combine this visual style with a practical purpose.

## Possible Widget Categories

| Category | Example Widget | Portfolio Skill |
| --- | --- | --- |
| Food | Restaurant mood picker | Recommendation logic |
| Travel | Trip countdown or packing helper | Planning workflow |
| Music | Playlist mood card | Personalization |
| Reading | Current book or learning tracker | Knowledge organization |
| Wellness | Habit reflection card | Metrics and reflection |
| AI | Prompt of the day | AI literacy and prompt design |
| Productivity | Focus intention widget | Decision support |

## Questions To Answer Next

1. Which hobbies feel most authentic to include?
2. What Pinterest style should guide the visual direction?
3. Should the first version be a web mockup or a native iPhone widget?
4. What would make each widget useful in daily life?
5. How can each widget show a skill that belongs in a portfolio?

## Research Output Goal

The research phase should produce:

- a written inspiration statement
- a visual style guide
- a list of possible widget concepts
- a short explanation of how AI supports the process
- a build plan for future mockups
